Prompts

This file contains the prompts used for the LLM comparison matrix.

Prompt 1: Technical Explanation

Explain the difference between RAM and storage in simple terms for a non-technical user.

Evaluation Focus
Factual accuracy
Simplicity
Clarity
Usefulness for a beginner
Prompt 2: Customer Support Reply

Write a polite customer support reply to a customer who was charged twice for a subscription and is frustrated because nobody responded yesterday.

Evaluation Focus
Tone
Empathy
Resolution clarity
Billing escalation
Avoiding unsupported claims
Prompt 3: Formatting Constraint

Summarize this in exactly three bullet points and do not add any extra text:

A strong password should be long, unique and difficult to guess. It should not include personal information. Password managers can help users create and store secure passwords.

Evaluation Focus
Exact instruction following
Formatting
Conciseness
No extra information
Prompt 4: Troubleshooting Checklist

My laptop connects to Wi-Fi, but websites are not loading. Give me a short troubleshooting checklist.

Evaluation Focus
Technical accuracy
Completeness
Practical order of steps
Clear escalation path
