Prompt 1: Technical Explanation
Response A

RAM and storage are basically the same because both keep information on your computer. If you have more RAM, you can save more photos, videos and files.

Response B

RAM is short-term memory that your computer uses while programs are open. Storage is long-term memory where your files, apps and operating system stay even after the computer turns off.

A simple way to think about it: RAM is like your desk while you work, and storage is like a cabinet where you keep things permanently.

Response C

RAM means random access memory and storage means the hard drive or SSD. RAM is volatile memory while storage is non-volatile memory. RAM is used by active processes and storage persists data across reboots.

Prompt 4: Troubleshooting Checklist
Response A

Restart your laptop and router. If it still does not work, contact your internet provider.

Response B

Here is a short checklist:

Check whether other devices can open websites on the same Wi-Fi.
Restart your browser and try a different website.
Clear DNS cache using ipconfig /flushdns on Windows.
Disable VPN, proxy or firewall temporarily to test.
Restart the router and reconnect to Wi-Fi.
If only this laptop is affected, reset the network adapter or update the Wi-Fi driver.

If none of these steps work, contact your ISP or IT support with the results.

Response C

Your Wi-Fi card is probably broken. You should replace the laptop’s wireless adapter or buy a new one.
